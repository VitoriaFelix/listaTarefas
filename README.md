## Clonar o repositório
```bash
$ git clone https://github.com/VitoriaFelix/listaTarefas.git
```
## Ir para o repositório
```bash
$ cd listaTarefas/app
```

## Instalar dependências
```bash
npm install
```

## Executar a aplicação
```bash
$ npm start
```
